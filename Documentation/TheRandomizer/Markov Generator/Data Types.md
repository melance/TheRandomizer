# Number
A numeric literal.

Examples
- 123
- 3234
- 4.33

# String
A text literal enclosed in quotes

Examples:
- "Elf"
- "Fighter"
- "Goodbye world"

# Boolean
A logical truth value

Values:
- true
- false

# Item List Name
A key that points to an `Item List`.

# Item List
A list of `Item`

# Item
A single selected value from a list.

Structure:
Content → String
Weight → Number
