Calculations

One of the most important features of the generators is the ability to run calculations.  This functionality is provided by the NCalc calculation engine which is extended in the randomizer generators with the following:


Parameters

Parameters defined for the generator are available in the calculations and can be referenced by their parameter name.


Functions

In addition to the standard functionality offered by NCalc, the following functions are available


Function

Description

Example

CBool (value)

Converts the provided value to a boolean.

CBool(1) Converts 1 to True

CDbl (value)

Converts the provided value to a double.

CDbl("0.1") Converts "0.1" to 0.1

CInt (value)

Converts the provided value to a 32-bit integer.

CInt("22") Converts "22" to 22

CLng (value)

Converts the provided value to a 64-bit integer.

CLng("1218189") Converts "1218189" to 1218189

Rnd ()

Generates a random non-negative integer.

Rnd() Generates a non-negative random integer

Rnd (max)

Generates a random non-negative integer that is less than or equal to max.

Rnd(42) Generates a number between 0 and 42 inclusive

Rnd (min, max)

Generates a random integer greater than min and less than or equal to max.

Rnd(1,6) Generates a number between 1 and 6 inclusive

Roll (sides)

Rolls one die with the number of sides provided.

Roll(20) Rolls 1d20

Roll (count, sides)

Rolls count dice with the number of sides provided.

Roll(3,6) Rolls 3d6

Roll (count, sides, modifier)

Rolls count dice with the number of sides provided and alters the result by the amount of the modifier.

Roll(3,6,-1) Rolls 3d6-1

Roll (count, sides, options[])

Rolls count dice with the number of sides provided using the options provided.  Options are detailed below.

Roll(4,6,"DL") Rolls 4d6 and drops the lowest

Roll (count, sides, modifier, options[])

Rolls count dice with the number of sides provided using the options provided and alters the result by the amount of the modifier.

Roll(2,20,+3,"DL") Rolls 2d20, drops the lowest, and adds 3

LastRoll ()

Returns the result of the last roll.

LastRoll() Returns the result of the last roll

LastRoll (detail)

Returns the detail of the last roll.  Detail is a string that can be: Result, ResultList, Successes, Failures, or Botches.

LastRoll("ResultList") Returns all die rolls from the last roll separated by commas

Pick (values[])

Selects one item from values at random.

Pick("Red","Green","Blue") Picks either "Red","Green", or "Blue"

UCase (value)

Converts value to uppercase

UCase("dog") Returns "DOG"

LCase (value)

Converts value to lowercase

LCase("DOG") Returns "dog"

TCase (value)

Converts value to titlecase

TCase("big RED dog") Returns "Big Red Dog"

ToOrdinal (number)

Returns the ordinal value of the provided number.

ToOrdinal(34) Returns "34th"

ToText (number)

Returns the word for the provided number.

ToText(123) Returns "One Hundred Twenty Three"

Format (string, values[])

Inserts the values into the string based on their index.

Format("The {0} is {1}", "Rose", "Red") Returns "The Rose is Red"

Generate (string)

Calls a generator with the name provided by string.

Generate("ElfName") Calls a generator called "ElfName"

Generate(string, <string, value>[])

Calls a generator with the name provided by string and provides the list of parameters in the array provided.  MaxLength can be included as a parameter in this list.

Generate("ElfName","MaxLength",20,"Gender","Male") Calls the generator called "ElfName" and passes in a Max Length of 20 and a parameter called "Gender" with the value "Male"


Roll Options

The roll function has several different options that allow it to handle most dice roll types.


Option

Name

Description

Variable

Sample

DH

Drop Highest

Drops the highest die roll(s)

Optional: Number of dice to drop

Roll(2,20,'DH') = 12,8; Result = 8

DL

Drop Lowest

Drops the lowest die roll(s)

Optional: Number of dice to drop

Roll(5,6,'DL',2) = 4,4,6,2,1; Result = 14

EX

Exploding Dice

Reroll any roll that is the highest value and add to the results.


Roll(3,6,'EX') = 4,4,6,6; Result = 20

CX

Compound Exploding Dice

Reroll any roll that is the highest value and add to the results. Continue until no highest value dice are rolled.


Roll(3,6,'EX') = 4,4,6,6,6,3; Result = 29

GT

Greater Than Target

Count any roll greater than the target number as a success.

Target Number

Roll(4,8,'GT',4) = 4,2,3,8; Successes = 1

GE

Greater Than or Equal to Target

Count any roll greater than or equal to the target number as a success.

Target Number

Roll(4,8,'GE',4) = 4,2,3,8; Successes = 2

LT

Less Than Target

Count any roll less than the target number as a success

Target Number

Roll(4,8,'LT',4) = 4,2,3,8; Successes = 2

LE

Less Than or Equal to Target

Count any roll less than or equal to the target number as a success

Target Number

Roll(4,8,'LE',4) = 4,2,3,8; Successes = 3

R1

Rule of One

Count 1's as botches


Roll(3,10,'GE',6,'R1') = 9,7,1; Successes = 2, Botches = 1

RA

Reroll Above

Reroll any roll above the target number and replace that roll

Target Number

Roll(4,6,'RA',5) = 2,3,6,5,3; Result = 10

RB

Reroll Below

Reroll any roll below the target number and replace that roll

Target Number

Roll(4,6,'RB',2) = 1,2,5,4,6; Result = 17
